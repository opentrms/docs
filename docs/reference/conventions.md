# Conventions: snake_case, system date, FX

Covers cross-cutting conventions: Jackson is configured for `SNAKE_CASE`
property naming on the wire, business logic is driven by a `SystemDate`
abstraction rather than the wall clock, and FX handling follows the backend's
`FX_Convention.md`.
